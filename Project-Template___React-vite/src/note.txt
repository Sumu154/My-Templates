1. main.jsx
Used to render the entire React app into the DOM. Connects the React app to index.
html using ReactDOM.createRoot.

2. index.css
Defines global styles for the app. Use it for base styles like 
fonts, resets, or shared styling.

3. App.jsx
The main component of your app. Contains the structure and logic for your
application.

4. App.css
Specific styles for the App.jsx component. Keeps component styling modular.
